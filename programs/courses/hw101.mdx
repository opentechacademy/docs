## Course Overview

HW101 is our flagship hardware course designed for learners with zero prior experience. Through guided readings, quick checks, and physical video walkthroughs, you will learn to identify all major desktop computer components, understand their functions, and observe how they work together as a single system.

---

## Course Outline

### Module 1: CPU and Motherboard
Start with the processor and the main board, then connect the vocabulary to a real desktop layout with a guided video walkthrough.
- **1.1 What the CPU does**: Understand the processor as the part that handles instructions and helps direct system activity.
- **1.2 What the motherboard does**: See the motherboard as the main board that lets the major parts connect to each other.
- **1.3 Watch: Inside a Computer**: Video walkthrough mapping the motherboard, CPU, RAM, storage, and power supply in a physical desktop PC.
- **1.4 Quiz: CPU and motherboard basics**: 5-question check covering CPU function, motherboard slots, and platform compatibility.

### Module 2: RAM and Storage
Understand the difference between active working memory and long-term data storage, then connect those ideas to everyday speed and responsiveness.
- **2.1 What RAM does**: Learn why RAM is temporary working memory while the computer is in use.
- **2.2 What storage does**: Understand where the operating system, apps, and saved files live long-term.
- **2.3 Quiz: RAM and storage basics**: 5-question check comparing volatile memory (RAM) and non-volatile storage (SSD/HDD) performance and latency.

### Module 3: Graphics, Ports, and Expansion
Learn how the system handles visuals, how peripherals connect, and how to read the back panel with more confidence.
- **3.1 What a GPU does**: Understand the graphics processor as the part that helps create what you see on the display.
- **3.2 What ports and expansion cards do**: Learn how the computer connects to accessories and how extra features can be added.
- **3.3 Watch: PC ports and back-panel connections**: Video walkthrough identifying USB, display, audio, Ethernet, and other common connections on a real system back panel.
- **3.4 Quiz: Graphics and connectivity**: 5-question check on dedicated GPUs, expansion slots, USB ports, and wired Ethernet links.

### Module 4: Power, Cooling, and the Case
See how the system gets power, stays cool, and stays physically organized so parts can operate safely and reliably.
- **4.1 What the power supply does**: Understand the power supply as the part that feeds electricity to the rest of the computer.
- **4.2 What cooling and the case do**: Learn how fans, heatsinks, and the case help protect the system from heat and physical damage.
- **4.3 Quiz: Power and cooling basics**: 5-question check on PSU voltage conversion, thermal protection, case airflow design, and mounting components.

### Module 5: How the Parts Work Together
Bring the parts together with a startup story, a physical teardown, and one final role-based review of the full system.
- **5.1 What happens when you turn on a computer**: Follow a simple power-on story from the power button to the operating system loading.
- **5.2 Part roles review**: Review the role of each main part in one simple end-to-end example.
- **5.3 Watch: What each PC part does in a beginner build**: Real hardware video walkthrough demonstrating how parts interact during installation.
- **5.4 Final Quiz: What each part does**: Cumulative assessment testing the basic functions of all major PC components.
